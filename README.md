# Notas:
App de consola interactiva, para generar un listado de tareas, modificarlas o borrarlas.

```

=======================
 Seleccione una opción 
=======================

? ¿Qué desea hacer? (Use arrow keys)
❯ 1. Crear tarea 
  2. Listar tareas 
  3. Listar tareas completadas 
  4. Listar tareas pendientes 
  5. Completar tarea(s) 
  6. Borrar tarea 
  0. Salir 

```
